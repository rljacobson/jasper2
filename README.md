Required setup:

```shell script
gem install jekyll bundler
bundle update
pip install pygments-mathematica

```

To build and serve locally:
```shell script
bundle exec jekyll serve
```

To build for deployment:
```shell script
JEKYLL_ENV=production bundle exec jekyll build
```
